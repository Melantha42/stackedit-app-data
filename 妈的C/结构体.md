
# 结构体

**结构体**是将不同的数据类型整合为一个结构的集合，使用“struct”函数表达。
 
 >- eg:
  struct  身法//（结构名）
  {
int   身法种类数量；
char   身法名称[ 10 ];
 double 身法使用率；
 //类型名 结构成员名；
  } ；

如上便定义了一个结构体。

若要在主调函数中使用已经定义的结构体，只需引用结构名：
如 main()

结构是C语言中一种新的**构造数据类型**，它能使有内在关系的**不同类型的数据**统一为一个*整体*，使其相互关联。

结构体变量：

>*若数组元素中每一个都是**结构体**，则称其为**结构体数组**。*
