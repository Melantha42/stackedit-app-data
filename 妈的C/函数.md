
**就他妈主调函数外写一个关于实参x/y的被调函数表达式，或是对x,y进行一系列的操作，包括不限于通过定义被调函数以数组的方式向主调函数输出指定值、或者内部再次嵌套新函数......**
  
>其中实参可以是变量X/Y；
也可以是常量C1/C2；
也可以是表达式x+y,x-y；
也可以是函数 add(2,3),3；
也可以是指针地址&a ,&b；
也可以是数组 int a[ ], int b[ ]；

   当实参内元素采用指针地址时，通常是要达成修改主调函数中的变量或高效传输大量数据的目的。
    
     incase（修改主调函数变量）:
     #include<stdio.h>
     void function(int *p, int *q)
     { int t;
     t=*p;
     *q=*p;
     *q=t;
     }
     int main (void){
     int a=4;
     int b=8;
     function(&a,&b); 
     //此处&a,&b意在将变量x与y的地址传入function；
     printf("%d,%d",a,b);
     //得到输出8
     return 0;
     }
