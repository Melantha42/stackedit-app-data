


# 时间单位
1 秒 = 1×10$^3$ 毫秒 = 1×10$^6$ 微秒 （千进制）

# 存储容量
- 1KB = 2$^1$$^0$B= 1024B = 8 × 1024 bit
- 1MB = 2$^1$$^0$KB = 2$^2$$^0$B
- 1GB = 2$^1$$^0$MB = 2$^2$$^0$KB = 2$^3$$^0$B
- 1TB = 2$^1$$^0$GB = 2$^2$$^0$MB = 2$^3$$^0$KB = 2$^4$$^0$B
