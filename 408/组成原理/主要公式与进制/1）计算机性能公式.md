
### CPI：执行一条指令平均所需的时钟周期数
#### ——CPI=总时钟周期数/总指令数

### MIPS(Million Instruction Per Second):百万条指令每秒，即单位时间内执行的指令数
#### ——MIPS=指令数/执行时间×10$^6$
联系：MIPS= f(时钟频率)/CPI×10$^6$

# 公式理解
-   f = 时钟频率（Hz，即每秒周期数）
    
-   T = CPU 执行时间（秒）
    
-   N = 指令总数
    
-   C = 总时钟周期数

CPI=C/N
C=N×CPI

T=C/f
