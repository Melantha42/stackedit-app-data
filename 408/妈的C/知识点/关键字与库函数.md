

*   **关键字**是C语言的**核心命令和基础规则**，是编译器直接认识的“内部指令”。
*   **库函数**是别人用C语言（结合关键字）写好的**现成工具集**，是“外部工具包”。
### 核心区别对比表

| 特征 | 关键字 | 库函数 |
| :--- | :--- | :--- |
| **本质** | 语言的**基本组成部分**，是“语法本身” | 别人预先编写好的**函数**，是“功能的实现” |
| **来源** | 由C语言标准定义，是**编译器内置**的 | 由标准库（如glibc）或第三方库**提供** |
| **数量** | 固定且数量少（C89有32个，C99等后续标准略有增加） | 数量非常多，并且可以无限扩展 |
| **使用方式** | 直接使用，例如：`int a;`, `if (x) {...}` | 需要包含相应的**头文件**，例如：`#include <stdio.h>` |
| **类比** | 英语中的**语法规则和核心词汇**（如：is, the, if, and） | 英语中的**成语或经典句子**（如：“Break a leg!”） |
| **例子** | `int`, `if`, `for`, `while`, `return`, `char` | `printf()`, `scanf()`, `strlen()`, `malloc()` |



# 考研重点关键字

以下是必须熟练掌握的**核心关键字**，它们构成了C程序的基本逻辑框架。

#### 1. 数据类型相关
- **`int`, `char`, `float`, `double`**：基本数据类型，最常考的是它们之间的**转换（隐式/显式）**、**取值范围**和**占用内存大小**（`sizeof`操作符）。
- **`void`**：用于表示“无类型”，常用于函数参数和返回值。
- **`const`**：定义常量，理解其“只读”特性。常与指针结合考查（如 `const int *p` vs `int * const p`）。

#### 2. 复杂类型构造
- **`struct`**：**重中之重**。用于构建复杂数据结构（如链表、树的学生信息等）。必须掌握结构体的定义、访问、以及结构体指针的使用。
- **`union`**：共用体，考查其所有成员共享内存的特点。
- **`enum`**：枚举类型，理解其本质是整型常量。

#### 3. 存储类别和作用域
- **`static`**：**超级重点**。考查其两种用法：
    1.  **修饰局部变量**：改变变量的生命周期，使其在函数调用间保持值。
    2.  **修饰全局变量和函数**：限制其作用域仅限于本文件。
- **`extern`**：声明外部变量/函数，用于多文件编程。

#### 4. 流程控制
- **`if`, `else`**：条件判断。
- **`switch`, `case`, `default`**：多分支选择。常考`break`的作用（防止case穿透）。
- **`for`, `while`, `do...while`**：循环。重点是**循环条件**、**循环体**和**循环控制变量**的变化。
- **`break`, `continue`**：循环控制。`break`跳出当前循环，`continue`跳过本次循环剩余部分。

#### 5. 跳转和返回
- **`return`**：从函数返回值。考查其在`void`函数中的用法，以及函数返回值的使用。
- **`goto`**：虽然不推荐，但在某些特定场景（如跳出多层循环）可能考查。

#### 6. 指针和内存
- **指针本身不是关键字**，但`*`（解引用）和`&`（取地址）操作符与上述关键字紧密结合，是考研的核心之核心。
    - 理解 `int *p`, `int **p` 的区别。
    - 理解指针与数组的关系。
    - 理解指针与函数（函数指针）。

#  二、考研重点库函数

考研中对于库函数的考查，主要集中在**输入输出、字符串处理、内存管理和数学计算**这几个方面，要求能熟练使用它们来解决问题。

#### 1. 标准输入/输出函数 (`<stdio.h>`)
- **`printf`**：格式化输出。**必须熟练掌握格式控制符**，如 `%d`, `%f`, `%c`, `%s`, `%p`，以及宽度、精度控制。
- **`scanf`**：格式化输入。**重点注意**：`&`符号的使用，以及输入缓冲区问题。
- **`getchar()`, `putchar()`**：字符输入/输出。常用于处理缓冲区残留字符或简单菜单选择。
- **`gets()` (已废弃)， `fgets()`**：字符串输入。考研中应注意`gets`的安全性问题，建议使用`fgets`。
- **`sprintf()`, `sscanf()`**：用于字符串的格式化，非常实用。

#### 2. 字符串处理函数 (`<string.h>`)
- **`strlen`**：求字符串长度（不包含`\0`）。
- **`strcpy` / `strncpy`**：字符串拷贝。注意`strcpy`的安全风险。
- **`strcat` / `strncat`**：字符串连接。
- **`strcmp` / `strncmp`**：字符串比较。**牢记返回值含义**（`<0`, `0`, `>0`）。
- **`strstr`**：在字符串中查找子串。
- **`strchr`**：在字符串中查找字符。

#### 3. 内存管理函数 (`<stdlib.h>`)
- **`malloc`**：**动态内存分配的核心**。考查其用法 `(int *)malloc(n * sizeof(int))` 以及**返回值判断**（可能返回NULL）。
- **`free`**：释放动态分配的内存。考查**内存泄漏**问题，以及释放后应将指针置为`NULL`的好习惯。
- **`calloc`**：分配并初始化内存为零。
- **`realloc`**：调整已分配内存块的大小。

#### 4. 数学函数 (`<math.h>`)
- **`abs`, `fabs`**：取绝对值。
- **`sqrt`**：开平方。
- **`pow`**：幂运算。
- **`ceil`, `floor`**：向上/向下取整。

#### 5. 字符分类/转换函数 (`<ctype.h>`)
- **`isalpha`**, **`isdigit`**, **`isalnum`**：判断字符类型。
- **`toupper`**, **`tolower`**：字母大小写转换。
