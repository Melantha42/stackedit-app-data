




| 数据类型 | printf格式符 | scanf格式符 | 备注 |
|----------|--------------|-------------|------|
| int | `%d`, `%i` | `%d`, `%i` | 有符号整数 |
| unsigned int | `%u` | `%u` | 无符号整数 |
| short | `%hd` | `%hd` | 短整数 |
| long | `%ld` | `%ld` | 长整数 |
| long long | `%lld` | `%lld` | 长长整数 |
| float | `%f`, `%e`, `%g` | `%f` | 单精度浮点 |
| double | `%lf`, `%le`, `%lg` | `%lf` | 双精度浮点 |
| long double | `%Lf`, `%Le`, `%Lg` | `%Lf` | 长双精度 |
| char | `%c` | `%c` | 字符 |
| 字符串 | `%s` | `%s` | 字符串 |
| 指针 | `%p` | - | 地址 |
| 八进制 | `%o` | `%o` | 无符号八进制 |
| 十六进制 | `%x`, `%X` | `%x` | 无符号十六进制 |

这个大全涵盖了C语言中几乎所有的格式说明符，建议收藏作为参考！
