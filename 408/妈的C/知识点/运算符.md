

## 优先级顺序计算
-   在C语言中，乘法和除法是**左结合**的，优先级相同
    
-   `a / b * c` 等价于 `(a / b) * c`，而不是 `a / (b * c)`

## 三目运算符
>条件 ? 表达式1 : 表达式2;
 
 ### 示例：
>int max = (a > b) ? a : b;
>比较a与b，并输出其中最大的，



# 运算符总览

C语言的运算符主要分为以下几类：

1.  **算术运算符**
2.  **关系运算符**
3.  **逻辑运算符**
4.  **位运算符**
5.  **赋值运算符**
6.  **条件运算符（三目运算符）**
7.  **逗号运算符**
8.  **sizeof 运算符**
9.  **指针运算符**
10. **地址运算符**
11. **成员运算符**
12. **下标运算符**

---

### 1. 算术运算符

用于基本的数学运算。

| 运算符 | 名称       | 作用                             | 示例              | 备注                                                                 |
| :----- | :--------- | :------------------------------- | :---------------- | :------------------------------------------------------------------- |
| `+`    | 加法       | 两个操作数相加                   | `a + b`           |                                                                      |
| `-`    | 减法       | 两个操作数相减                   | `a - b`           |                                                                      |
| `*`    | 乘法       | 两个操作数相乘                   | `a * b`           |                                                                      |
| `/`    | 除法       | 两个操作数相除                   | `a / b`           | **整数相除结果为整数**，如 `5 / 2` 结果是 `2`，不是 `2.5`。           |
| `%`    | 取模（求余） | 返回除法后的余数                 | `a % b`           | **只能用于整数**。结果的符号与被除数 (`a`) 相同。如 `-5 % 2` 结果是 `-1`。 |
| `++`   | 自增       | 操作数的值增加1                  | `a++` 或 `++a`    | **考点：前缀与后缀的区别**。`a++`（后缀）先使用a的值，再自增；`++a`（前缀）先自增，再使用a的值。 |
| `--`   | 自减       | 操作数的值减少1                  | `a--` 或 `--a`    | 同上。                                                               |

**考研重点**：
*   `/` 运算符的整数除法特性。
*   `%` 运算符的整数限制和结果的符号问题。
*   `++` 和 `--` 的前缀/后缀区别，尤其是在复杂表达式中的求值顺序和副作用，例如 `int i = 1; int j = (i++) + (++i);` 这种代码是**未定义行为**，不同编译器结果可能不同，考研中应避免。

---

### 2. 关系运算符

用于比较两个值，结果为 `1` (真) 或 `0` (假)。

| 运算符 | 名称     | 作用             | 示例     |
| :----- | :------- | :--------------- | :------- |
| `==`   | 等于     | 判断是否相等     | `a == b` |
| `!=`   | 不等于   | 判断是否不相等   | `a != b` |
| `>`    | 大于     | 判断是否大于     | `a > b`  |
| `<`    | 小于     | 判断是否小于     | `a < b`  |
| `>=`   | 大于等于 | 判断是否大于等于 | `a >= b` |
| `<=`   | 小于等于 | 判断是否小于等于 | `a <= b` |

**考研重点**：
*   注意 `==` (相等判断) 和 `=` (赋值) 的区别，这是常见的错误来源。
*   关系运算符的优先级低于算术运算符。例如 `a + b > c - d` 等价于 `(a + b) > (c - d)`。

---

### 3. 逻辑运算符

用于连接多个条件，结果为 `1` (真) 或 `0` (假)。

| 运算符 | 名称 | 作用                                   | 示例              | 备注                                                                                                 |
| :----- | :--- | :------------------------------------- | :---------------- | :--------------------------------------------------------------------------------------------------- |
| `&&`   | 逻辑与 | 两个条件都为真时结果为真             | `a && b`          | **短路求值**：如果第一个操作数为假，则不再计算第二个操作数。                                           |
| `\|\|` | 逻辑或 | 两个条件至少一个为真时结果为真       | `a \|\| b`        | **短路求值**：如果第一个操作数为真，则不再计算第二个操作数。                                           |
| `!`    | 逻辑非 | 逆转操作数的逻辑状态，真变假，假变真 | `!a`              | 一元运算符。                                                                                         |

**考研重点**：
*   **短路求值**是核心考点。例如 `(a != NULL) && (a->data == 10)`，如果 `a` 为 `NULL`，则不会访问 `a->data`，避免了程序崩溃。
*   优先级： `!` > `&&` > `||`。

---

### 4. 位运算符

直接对整数在内存中的二进制位进行操作。

| 运算符 | 名称     | 作用                                       | 示例        | 备注                                             |
| :----- | :------- | :----------------------------------------- | :---------- | :----------------------------------------------- |
| `&`    | 按位与   | 两个位都为1时，结果才为1                 | `a & b`     | 常用于将特定位清0（掩码）。                      |
| `\|`   | 按位或   | 两个位有一个为1时，结果就为1             | `a \| b`    | 常用于将特定位置1。                              |
| `^`    | 按位异或 | 两个位相异时，结果为1                   | `a ^ b`     | 相同为0，不同为1。常用于翻转特定位。             |
| `~`    | 按位取反 | 对操作数的每一位取反（0变1，1变0）      | `~a`        | 一元运算符。                                     |
| `<<`   | 左移     | 将二进制位全部左移若干位，高位丢弃，低位补0 | `a << n`    | 相当于乘以 \(2^n\)。                             |
| `>>`   | 右移     | 将二进制位全部右移若干位                 | `a >> n`    | 对于**无符号数**，高位补0；对于**有符号数**，高位补符号位（算术右移）。相当于除以 \(2^n\)（向下取整）。 |

**考研重点**：
*   区分 `&`、`|` 和 `&&`、`||`。前者是按位操作，后者是逻辑操作。
*   移位运算的规则，特别是右移对于有符号数和无符号数的区别。
*   位运算常用于状态压缩、权限管理、优化计算等场景。

---

### 5. 赋值运算符

用于给变量赋值。

| 运算符 | 示例      | 等价于      |
| :----- | :-------- | :---------- |
| `=`    | `a = b`   |             |
| `+=`   | `a += b`  | `a = a + b` |
| `-=`   | `a -= b`  | `a = a - b` |
| `*=`   | `a *= b`  | `a = a * b` |
| `/=`   | `a /= b`  | `a = a / b` |
| `%=`   | `a %= b`  | `a = a % b` |
| `&=`   | `a &= b`  | `a = a & b` |
| `\|=`  | `a \|= b` | `a = a \| b` |
| `^=`   | `a ^= b`  | `a = a ^ b` |
| `<<=`  | `a <<= n` | `a = a << n` |
| `>>=`  | `a >>= n` | `a = a >> n` |

---

### 6. 条件运算符 (三目运算符)

`表达式1 ? 表达式2 : 表达式3`
如果 `表达式1` 为真，则整个表达式的值为 `表达式2` 的值；否则为 `表达式3` 的值。
**作用**：简单的条件判断和赋值。
**示例**：`max = (a > b) ? a : b;` // 将a和b中的较大值赋给max。

---

### 7. 逗号运算符

`表达式1, 表达式2, ..., 表达式n`
从左到右依次计算各个表达式，**整个逗号表达式的值和类型是最后一个表达式的值和类型**。
**作用**：主要用于for循环的多个控制变量。
**示例**：`for(i=0, j=10; i<j; i++, j--)`

---

### 8. sizeof 运算符

`sizeof(类型 或 表达式)`
返回**类型**或**表达式结果类型**所占的字节数。
**注意**：它是**运算符**，不是函数。在编译时求值。
**示例**：
```c
int a;
sizeof(int); // 返回4（在32/64位系统中通常是4）
sizeof(a);   // 返回4
sizeof a;    // 正确，可以不加括号（用于变量时）
```

---

### 9. 指针运算符与地址运算符

| 运算符 | 名称     | 作用                     | 示例     |
| :----- | :------- | :----------------------- | :------- |
| `*`    | 解引用   | 获取指针所指向内存的值 | `*p`     |
| `&`    | 取地址   | 获取变量的内存地址     | `&a`     |

---

### 10. 成员运算符

| 运算符 | 名称       | 作用                     | 示例      |
| :----- | :--------- | :----------------------- | :-------- |
| `.`    | 直接成员选择 | 访问结构体或共用体的成员 | `s.member` |
| `->`   | 间接成员选择 | 通过指针访问结构体或共用体的成员 | `p->member` (等价于 `(*p).member`) |

---

### 11. 下标运算符

`数组名[索引]`
用于访问数组元素。`a[i]` 在编译时等价于 `*(a + i)`。

---

### 运算符优先级与结合性总结（重要！）

这是考研选择题和大题中分析复杂表达式的关键。记住大致规律即可，不需要死记硬背，不确定时多用括号。

**从高到低大致顺序**：

1.  **初等运算符**：`()` `[]` `.` `->` `(类型){初始化列表}` (C99)
2.  **单目运算符**：`!` `~` `++` `--` `+`(正) `-`(负) `*`(解引用) `&`(取址) `sizeof` `_Alignof`
    *   **结合性：从右到左**
3.  **算术运算符**： `*` `/` `%` > `+` `-`
4.  **移位运算符**： `<<` `>>`
5.  **关系运算符**： `<` `<=` `>` `>=` > `==` `!=`
6.  **位运算符**： `&` > `^` > `|`
7.  **逻辑运算符**： `&&` > `||`
8.  **条件运算符**： `? :`
    *   **结合性：从右到左**
9.  **赋值运算符**： `=` `+=` `-=` `*=` `/=` `%=` `<<=` `>>=` `&=` `^=` `|=`
    *   **结合性：从右到左**
10. **逗号运算符**： `,`

**记忆口诀**：**单算移关与，异或逻条赋**。（非绝对精确，但覆盖大部分）

**给考研学子的建议**：
1.  **理解为主，记忆为辅**：理解每个运算符的本质，特别是那些有“坑”的（如 `/`, `%`, `++`）。
2.  **勤用括号**：在写复杂表达式时，使用括号明确优先级，提高代码可读性，避免错误。
3.  **多做练习**：找一些经典的运算符相关的考研真题和模拟题练习，特别是分析程序输出结果的题目。
