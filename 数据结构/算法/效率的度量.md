

# 算法效率
## 时间复杂度
事先预估算法的**时间开销T（n）**与**问题规模n**的关系
>通常来说，当算法运行的问题足够大（n→$\infty$ ）时，时间复杂度遵守
>- 加法规则：
> T（n）=T$_1$（n）＋T$_2$（n）=O（f(n)）＋O（g(n)）=O（max(f(n),g(n))）
>【即：多项相加，只保留最高阶项，且系数变为1（高阶无穷）】
>- 乘法规则：
>T（n）=T$_1$（n）$×$T$_2$（n）=O（f(n)）×O（g(n)）=O（f(n)×g(n)）
